## 📞 Common Questions Answered

**Q: Where do I start?**  
A: Read `EXECUTIVE_SUMMARY.md` first, then `README.md` in development_lifecycle folder.

**Q: How long will this really take?**  
A: 16 weeks with 2 developers (1 backend, 1 frontend). Solo: 6 months.

**Q: Can I skip phases?**  
A: No. Each phase builds on previous. But you can compress Phase 5 if confident in quality.

**Q: What if I'm stuck on satellite data?**  
A: Phase 0 includes testing. If Earth Engine fails, use USGS EarthExplorer (fallback documented in Phase 1).

**Q: How do I validate this really works?**  
A: Phase 5 includes real-world testing with 2+ sample Indian projects.

**Q: Will this integrate with carbon registries?**  
A: Phase 7 covers this. MVP (Phases 1-5) focuses on evidence generation.

## 🔧 Tech Stack (All Proven, Open Source)

**Backend:** Python 3.9+, FastAPI, SQLAlchemy  
**Frontend:** React, Material-UI, Leaflet  
**Database:** PostgreSQL + PostGIS  
**Cloud:** Microsoft Azure  
**Satellite Data:** Google Earth Engine API  
**ML:** scikit-learn, XGBoost  
**DevOps:** GitHub Actions, Docker
